1. DESCRIPTION
-------------------------------------------------------------------------------
Set of batch scripts for svn command line tools including svn command line tools
(Win32) as is from:
* TortoiseSVN
* CollabNet
* SlikSVN
* Cygwin
* VisualSVN
* win32svn
